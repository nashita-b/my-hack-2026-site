## 1. Key Decisions (locked so far)

| Decision | What we decided |
|---|---|
| Judging format | Walkthrough at team tables, not room presentations |
| Judge grouping | 8 judges → 4 pairs, each pair covers 3–4 teams |
| Scoring | Average of both judges' scores per team, per category |
| Tiebreak rule | 1) Higher Impact & Relevance score wins. 2) If still tied, the judge pair that saw both teams decides together. 3) Only escalate to organizers if the tie is cross-pair. |
| Prize structure | $2,000 total — High School and College divisions judged separately, each awarding 1st ($500) / 2nd ($300) / 3rd ($200) |
| Tracks | Inspiration only — not a formal category, not scored |
| HS/college split | Confirmed by admin — NOT split by track, split by division for prizes only |
| SSL hours | Self-reported to Nashita post-event |
