Build resources

Free tiers exist for all of these — none are required, but they're what past teams have leaned on to build fast.

Stuck on an idea?
Claude
Talk through a problem statement and narrow down a real, buildable idea.
claude.ai
ChatGPT
Brainstorm angles on a track and stress-test your idea's feasibility.
chatgpt.com
Miro
Free collaborative whiteboard — great for mapping ideas as a team.
miro.com
AI & ML
OpenAI
Versatile LLM API for chatbots and NLP tasks.
platform.openai.com
Gemini API
Google's GenAI API with search-integrated answers.
ai.google.dev
Ollama
Run AI models locally — good for offline, private use.
ollama.com
MCP
Multi-context prompting — improve AI reasoning across steps.
github.com/jina-ai/dev-gpt
A2A
Agent-to-agent — build chains of agents for multi-step workflows.
github.com/jina-ai/a2a
Frontend
Claude
Generate frontend code instantly with a web editor.
claude.ai
Bolt.new
React-based scaffolding, fast to prototype in.
bolt.new
Lovable
No-code UI builder, easy to prototype with.
lovable.dev
Cursor
AI-enabled code editor with strong assist.
cursor.so
v0
Design-first AI UI builder from Vercel.
v0.dev
Hosting & backend
Vercel
Full-stack hosting, great for frontend + backend.
vercel.com
Firebase
Real-time database and auth, good for web/mobile sync.
firebase.google.com
Supabase
Open-source Firebase alternative — SQL + REST built in.
supabase.com
Netlify
Simple, free static hosting — drag and drop to deploy.
netlify.com
Mobile & automation
Flutter
Google's UI toolkit for cross-platform native apps.
flutter.dev
n8n
Open-source automation — connect apps and run workflows.
n8n.io
crewAI
AI agent orchestrator — build task and process agents.
github.com/joaomdmoura/crewAI
